# Threads Affiliate Intelligence System

Sistem otomasi cerdas untuk memantau platform Threads, mendeteksi niat (intent) pengguna menggunakan AI (Gemini), menemukan produk yang relevan, dan menghasilkan balasan afiliasi Shopee secara otomatis.

## Arsitektur & Teknologi
*   **Bahasa Pemrograman:** Python 3.12+
*   **Browser Automation:** Playwright
*   **Web Framework:** Flask
*   **Database:** SQLite
*   **AI Engine:** Google Gemini API
*   **Configuration:** `python-dotenv`

## Persyaratan Sistem
- Python 3.12+
- Browser dependencies (via Playwright)
